## DevC++

### 创建项目文件夹

随便找个地方建一个文件夹

### 写HelloWorld

```c++
// 1.cpp
#include<iostream>
using namespace std;
int main(){
    cout << "HelloWorld" << endl;
    return 0;
}
```

然后编译

```bash
g++ 1.cpp -o 1.exe
```

运行

```bash
./1.exe
```
